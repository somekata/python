第4章: ループ処理

ループ処理は、同じ処理を繰り返し実行するために使われる重要なプログラミング技術です。この章では、Pythonにおけるループ処理の基本を学びます。

forループの基本

`for`ループは、指定した範囲内のデータを順番に処理するために使用されます。たとえば、リスト内のすべての要素を順に処理するには次のようにします。 

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

このコードはリスト`fruits`の中の要素を1つずつ取り出し、`print`関数で表示します。

range()関数を使ったループ

`range()`関数を使うことで、指定した回数分だけループを繰り返すことができます。 

for i in range(5):
    print(i)

このコードは、`0`から`4`までの整数を順に表示します。`range(5)`は`0`から始まり、`5`未満の整数を生成することを意味します。

whileループ

`while`ループは、条件が`True`である間、繰り返し処理を実行します。次の例では、`x`が5より小さい間、ループが実行されます。 

x = 0

while x < 5:
    print(x)
    x += 1

このループでは、`x`が`0`から始まり、毎回1ずつ増加し、`x`が`5`未満である限り繰り返し処理されます。

ループの終了とスキップ

`break`と`continue`を使って、ループを途中で終了したり、特定の処理をスキップしたりすることができます。 

for i in range(10):
    if i == 5:
        break  # ループを終了
    print(i)

for i in range(10):
    if i % 2 == 0:
        continue  # 偶数をスキップ
    print(i)

最初のループでは、`i`が`5`になった時点でループが終了します。2つ目のループでは、偶数がスキップされ、奇数だけが表示されます。
