第7章: matplotlibを使ったグラフ作成

この章では、Pythonの強力なグラフ作成ライブラリである matplotlib を使って、さまざまな種類のグラフを作成する方法を学びます。

基本的な折れ線グラフ

`matplotlib`で最も基本的なグラフは折れ線グラフです。次の例は、2つのリストを使ってシンプルな折れ線グラフを作成します。 

import matplotlib.pyplot as plt

# データ
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# グラフの作成
plt.plot(x, y)
plt.title('折れ線グラフ')
plt.xlabel('X軸')
plt.ylabel('Y軸')
plt.show()

このコードでは、リスト `x` と `y` のデータを使用して、折れ線グラフを描画しています。`plt.plot()`関数を使ってプロットし、`plt.show()`でグラフを表示します。

散布図の作成

散布図は、2つの変数の関係を視覚化するのに適しています。次のコードは、ランダムなデータを使った散布図の例です。 

import matplotlib.pyplot as plt

# データ
x = [1, 2, 3, 4, 5, 6]
y = [5, 7, 4, 6, 8, 7]

# 散布図の作成
plt.scatter(x, y)
plt.title('散布図')
plt.xlabel('X軸')
plt.ylabel('Y軸')
plt.show()

このコードは、`plt.scatter()`を使って散布図を作成し、2つの変数間の関係を視覚化します。

棒グラフの作成

棒グラフは、カテゴリー間の比較を行うのに適しています。次のコードでは、簡単な棒グラフを作成します。 

import matplotlib.pyplot as plt

# データ
categories = ['A', 'B', 'C', 'D']
values = [3, 7, 5, 9]

# 棒グラフの作成
plt.bar(categories, values)
plt.title('棒グラフ')
plt.xlabel('カテゴリー')
plt.ylabel('値')
plt.show()

このコードは、`plt.bar()`を使って棒グラフを作成し、各カテゴリーの値を視覚化します。

ヒストグラムの作成

ヒストグラムは、データの分布を視覚化するのに役立ちます。次の例では、ランダムデータを使ったヒストグラムを作成します。 

import matplotlib.pyplot as plt
import numpy as np

# ランダムデータ
data = np.random.randn(1000)

# ヒストグラムの作成
plt.hist(data, bins=30)
plt.title('ヒストグラム')
plt.xlabel('値')
plt.ylabel('頻度')
plt.show()

このコードでは、`numpy`を使ってランダムデータを生成し、`plt.hist()`を使ってヒストグラムを作成します。
