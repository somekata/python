第6章: Pythonのライブラリ紹介

Pythonには、多様なタスクを効率的に行うためのライブラリが豊富に用意されています。この章では、データ操作、可視化、機械学習、Webアプリケーション開発などに役立つライブラリを紹介します。

1. pandas

pandas は、データフレームを使用して表形式データを扱うためのライブラリです。CSVファイルの読み書きやデータの操作が簡単にできます。 

import pandas as pd

# CSVファイルの読み込み
df = pd.read_csv('data.csv')
print(df.head())

2. numpy

numpy は、数値計算を効率的に行うためのライブラリで、多次元配列（ndarray）の操作に特化しています。 

import numpy as np

# 配列の作成
arr = np.array([1, 2, 3, 4])
print(arr)

3. matplotlib

matplotlib は、データの可視化に使用されるライブラリで、折れ線グラフ、棒グラフ、散布図など、さまざまなグラフを作成できます。 

import matplotlib.pyplot as plt

# データのプロット
plt.plot([1, 2, 3], [4, 5, 6])
plt.title('Line Plot')
plt.show()

4. seaborn

seaborn は、統計的データの可視化に特化したライブラリで、美しいグラフを簡単に作成できます。 

import seaborn as sns

# データセットの読み込み
df = sns.load_dataset('iris')

# ペアプロットの作成
sns.pairplot(df, hue='species')
plt.show()

5. csv（標準ライブラリ）

csv は、CSVファイルを読み書きするためのPython標準ライブラリです。pandasほど高度ではありませんが、シンプルなファイル操作に向いています。実際の開発では、pandasの使用が一般的です。 

import csv

# CSVファイルの読み込み
with open('data.csv', newline='') as csvfile:
    reader = csv.reader(csvfile)
    for row in reader:
        print(row)

6. tkinter

tkinter は、PythonでGUIアプリケーションを作成するための標準ライブラリです。簡単なデスクトップアプリケーションを作成できます。 

import tkinter as tk

# ウィンドウの作成
window = tk.Tk()
window.title("Tkinter Window")
window.mainloop()

7. Flask

Flask は、軽量なWebアプリケーションフレームワークで、APIや小規模なWebサイトを簡単に作成できます。 

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

if __name__ == '__main__':
    app.run()

8. scikit-learn

scikit-learn は、機械学習に特化したライブラリで、分類、回帰、クラスタリング、次元削減などをサポートしています。 

from sklearn.linear_model import LinearRegression

# モデルの作成と訓練
model = LinearRegression()
X = [[1], [2], [3], [4]]
y = [1, 2, 3, 4]
model.fit(X, y)

# 予測
predictions = model.predict(X)
print(predictions)

9. 次元削減技術（PCA, t-SNE, UMAP）

scikit-learn ライブラリでは、PCA（主成分分析）やt-SNEなどの次元削減技術が使用できます。また、UMAPは、別のライブラリumap-learnを使用して実行します。次元削減は、高次元データを可視化や解析しやすい低次元に変換するために利用されます。

PCA

from sklearn.decomposition import PCA

pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)
print(X_pca)

t-SNE

from sklearn.manifold import TSNE

tsne = TSNE(n_components=2)
X_tsne = tsne.fit_transform(X)
print(X_tsne)

UMAP（umap-learnライブラリ）

import umap

umap_model = umap.UMAP(n_components=2)
X_umap = umap_model.fit_transform(X)
print(X_umap)

10. その他のライブラリ

  • openpyxl: Excelファイルの読み書きに使用。
  • sqlite3: 軽量なデータベース管理をサポート。
  • requests: HTTPリクエストを送信するためのライブラリ。
  • beautifulsoup4: Webスクレイピング用のライブラリ。
  • Django: 大規模なWebアプリケーションのためのフレームワーク。
