第8章: CSVの取り込みと解析

CSVファイル（Comma-Separated Values）は、データをテキスト形式で保存する一般的な形式です。この章では、Pythonの pandas ライブラリを使ってCSVファイルを取り込み、データを解析する方法を学びます。

pandasを使ったCSVの読み込み

pandasライブラリを使うと、CSVファイルを簡単に読み込んで、表形式のデータとして扱うことができます。まず、CSVファイルを読み込んでみましょう。 

import pandas as pd

# CSVファイルの読み込み
df = pd.read_csv('data.csv')

# 最初の5行を表示
print(df.head())

このコードでは、`pd.read_csv()`を使ってCSVファイルを読み込み、データを`DataFrame`オブジェクトに格納しています。`df.head()`は、最初の5行を表示するための関数です。

データの基本操作

`pandas`を使うと、データのフィルタリングや統計情報の取得が簡単に行えます。次に、いくつかの基本操作を見ていきましょう。 

# 特定の列を表示
print(df['列名'])

# 統計情報の表示
print(df.describe())

# フィルタリング（条件に基づいたデータの抽出）
filtered_data = df[df['列名'] > 100]
print(filtered_data)

これらの例では、特定の列を表示したり、データの要約統計量を確認したり、条件に基づいてデータをフィルタリングする方法を示しています。

データの可視化

pandasとmatplotlibを組み合わせて、CSVデータをグラフで視覚化することも可能です。次のコードでは、データを可視化する簡単な例を示します。 

import matplotlib.pyplot as plt

# データのプロット
df['列名'].plot(kind='bar')
plt.title('棒グラフ')
plt.xlabel('X軸')
plt.ylabel('Y軸')
plt.show()

このコードでは、`pandas`の`plot`メソッドを使ってデータを棒グラフとして可視化しています。`matplotlib`と組み合わせることで、より柔軟なグラフ作成が可能です。

CSVファイルへの書き出し

解析したデータを再びCSVファイルとして保存することも簡単です。次のコードでは、`pandas`を使ってCSVファイルを書き出します。 

# データをCSVファイルに保存
df.to_csv('output.csv', index=False)

このコードでは、`df.to_csv()`メソッドを使って、`DataFrame`を新しいCSVファイルとして保存しています。`index=False`はインデックスをCSVに含めないオプションです。
