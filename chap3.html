第3章: 条件分岐

条件分岐は、プログラムの中で特定の条件が満たされた場合に異なる処理を実行するための重要な構文です。この章では、Pythonにおける条件分岐の基本を学びます。

if文の使い方

条件分岐の基本構文は、`if`文です。例えば、次のコードでは数値が10以上かどうかをチェックしています。 

x = 15

if x >= 10:
    print("xは10以上です")

この例では、変数 `x` が10以上であれば「xは10以上です」というメッセージが表示されます。Pythonではインデントが非常に重要で、`if`文に続く処理は必ずインデント（スペース4つ）で区切られます。

else文とelif文

`if`文に続いて、条件が満たされなかった場合に実行される`else`文や、複数の条件を扱うための`elif`文があります。 

x = 5

if x > 10:
    print("xは10より大きいです")
elif x == 10:
    print("xは10です")
else:
    print("xは10未満です")

この例では、`x` が10より大きい場合、等しい場合、そしてそれ以外の場合に応じて異なるメッセージが表示されます。

条件式の書き方

Pythonでは、条件式には以下のような比較演算子を使います。

  • `==` : 等しい
  • `!=` : 等しくない
  • `>` : より大きい
  • `>=` : 以上
  • `<` : より小さい
  • `<=` : 以下

また、条件式を組み合わせて使うこともできます。例えば、次のように複数の条件を`and`や`or`で組み合わせることができます。 

x = 7
y = 20

if x > 5 and y < 30:
    print("xは5より大きく、yは30より小さい")
